# 质点运动学预习笔记

## 1. 运动学：积分的初步运用
在已知加速度 $\vec{a}(t)$ 或速度 $\vec{v}(t)$ 的情况下，通过积分求取位置信息。

### 例 1.5
设一点电荷在电场中运动，若加速度为 $\vec{a} = -k(t - t_0)$（其中 $k > 0$ 为常数），已知 $t = t_0$ 时点的速度为 $v_0$，求任意时刻点的速度和位置。

* **求速度 $\vec{v}(t)$：**
    由加速度定义 $\vec{a} = \frac{d\vec{v}}{dt}$，进行变量分离并积分：
    $$\int_{v_0}^{v} d\vec{v} = \int_{t_0}^{t} -k(t - t_0) dt$$
    得速度公式：
    $$v = v_0 - \frac{1}{2}k(t - t_0)^2$$

* **求位置 $\vec{x}(t)$：**
    由速度定义 $\vec{v} = \frac{d\vec{r}}{dt}$，代入已知初值条件：
    $$\int_{r_0}^{r} d\vec{r} = \int_{t_0}^{t} \left[ v_0 - \frac{1}{2}k(t - t_0)^2 \right] dt$$
    得质点位置：
    $$r = r_0 + v_0(t - t_0) - \frac{1}{6}k(t - t_0)^3$$

## 2. 切向加速度与法向加速度
在曲线运动中，加速度 $\vec{a}$ 可以分解为切向分量和法向分量。

### 核心公式
$$\vec{a} = \vec{a}_t + \vec{a}_n = \frac{dv}{dt}\vec{e}_t + \frac{v^2}{\rho}\vec{e}_n$$

* **切向加速度 $a_t$：**
    $$a_t = \frac{dv}{dt}$$
    **物理意义**：反映**“速率”**大小的变化快慢。
* **法向加速度 $a_n$：**
    $$a_n = \frac{v^2}{\rho}$$
    **物理意义**：反映速度**“方向”**的变化快慢（其中 $\rho$ 为曲率半径）。

### 推导关键点
$$\frac{d\vec{e}_t}{dt} = \omega\vec{e}_n = \frac{v}{\rho}\vec{e}_n$$
**提示**：$d\vec{e}_t = d\theta \vec{e}_n$ 的核心原因在于垂直的加速度只改变方向。可以想象切向量像单位圆那样旋转；同理，法向量求导也能得到切向量。

## 3. 角量描述与圆周运动

### 基本定义
* **角速度**：$\omega = \frac{d\theta}{dt}$
* **角加速度**：$\beta = \frac{d\omega}{dt} = \frac{d^2\theta}{dt^2}$

### 线量与角量的关系
对于半径为 $R$ 的圆周运动：
* 速度（速率）：$v = \omega R$
* 切向加速度：$a_t = \beta R$
* 法向加速度：$a_n = \omega^2 R = \frac{v^2}{R}$

### 匀变速圆周运动方程
其形式与直角坐标系下的匀变速直线运动一致：
* $\omega = \omega_0 + \beta t$
* $\theta = \theta_0 + \omega_0 t + \frac{1}{2}\beta t^2$
* $\omega^2 - \omega_0^2 = 2\beta(\theta - \theta_0)$

### 角速度矢量 $\vec{\omega}$
* **方向**：按**“右手螺旋定则”**确定（右手四指指向质点前进方向弯曲，大拇指指向即为 $\vec{\omega}$ 方向）。
* **关系式**：$\vec{v} = \vec{\omega} \times \vec{r}$

---

## 4. 坐标系的选择与比较

* **自然坐标系 $(\vec{e}_t, \vec{e}_n)$**：具有更强的物理意义，在研究切向、法向受力或运动时更常用。单位向量可以直接写。
* **直角坐标系 $(\vec{i}, \vec{j})$**：题目要求求速度等矢量时，使用直角坐标系可以同时表示大小和方向，便于计算。

## 5. 运动学物理应用专题

### 案例：速度分解（关联速度）
在滑轮拉船模型中，实际运动的是船，应当对船的速度进行分解：
$$u = v \cos \theta$$
（其中 $u$ 是绳子收缩速率，$v$ 是船速）

### 案例：加速度的矢量运算法
1.  利用点积求切向加速度：
    $$a_t = \vec{a} \cdot \vec{e}_t = \vec{a} \cdot \frac{\vec{v}}{v} = \frac{\vec{a} \cdot \vec{v}}{v}$$
2.  求得 $a_t$ 后，可利用勾股定理求法向加速度：
    $$a_n = \sqrt{a^2 - a_t^2}$$

### 案例：阻力下的积分（变量分离法）
已知 $a = -kv^2$，初始条件 $t = 0, x = 0, v = v_0$。
* **求 $v(t)$：**
    $$\frac{dv}{dt} = -kv^2 \Rightarrow \int_{v_0}^{v} v^{-2} dv = -k \int_{0}^{t} dt$$
    解得：
    $$-\frac{1}{v} + \frac{1}{v_0} = -kt \Rightarrow v = \frac{v_0}{1 + kv_0 t}$$
* **求 $x(t)$：**
    对上述结果再次积分即可：
    $$x = \int_{0}^{t} \frac{v_0}{1 + kv_0 t} dt = \frac{1}{k} \ln(1 + kv_0 t)$$

## 6. 其他补充
* **伽利略变换**：$\vec{v} = \vec{v}' + \vec{u}$
    （其中 $\vec{v}$ 为绝对速度，$\vec{v}'$ 为相对速度，$\vec{u}$ 为牵连速度）
* **求导法则**：叉积和点积均满足乘积求导法则。
    $$\frac{d}{dt}(\vec{A} \cdot \vec{B}) = \frac{d\vec{A}}{dt} \cdot \vec{B} + \vec{A} \cdot \frac{d\vec{B}}{dt}$$
    $$\frac{d}{dt}(\vec{A} \times \vec{B}) = \frac{d\vec{A}}{dt} \times \vec{B} + \vec{A} \times \frac{d\vec{B}}{dt}$$
