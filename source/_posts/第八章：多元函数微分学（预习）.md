# 第八章：多元函数微分学
日期：2026年1月18日

## 一、 多元函数基础

### 1. 核心思想
所谓的“**求微分**”，本质上就是寻找那个最贴合原函数的“**线性模型**”。

### 2. 基础定义
* **自变量**：可以理解为是从 $n$ 元欧氏空间中取出的点 $P(x_1, x_2, ..., x_n) \in D \subset \mathbb{R}^n$。
* **邻域 (Neighborhood)**：
    * $U(P_0, \delta)$ **邻域**：距离点 $P_0$ 小于 $\delta$ 的所有点。
      $$\rho(P_0, P) < \delta \iff \sqrt{(x - x_0)^2 + (y - y_0)^2} < \delta$$
    * $\mathring{U}(P_0, \delta)$ **空心邻域**：$\delta$ 邻域中去掉中心点 $P_0$。
      $$0 < \rho(P_0, P) < \delta$$

### 3. 点集与区域分类
* **点分类**：
    * **内点**：邻域完全包含在集合 $E$ 内。
    * **外点**：邻域完全包含在集合 $E$ 之外。
    * **边界点**：任意邻域内既含 $E$ 中的点，又含 $E$ 外的点（边界点可能属于 $E$，也可能不属于）。
* **集分类**：
    * **开集**：每一点都是内点（即 $\text{int } E = E$）。
    * **闭集**：其余集 $\mathbb{R}^n - E$ 是开集。
* **区域**：连通的开集称为区域（对应一元函数的区间概念）。

### 4. 极限与连续
* **趋近方式**：二元函数趋近点 $P_0$ 的方式有无穷多种。
* **判别方法**：
    * “**两路验证**”法：选取 $y = kx$ 或 $y = x^2$ 等不同路径逼近。如果不同路径的极限不相等，则**二重极限不存在**。
* **二重极限与累次极限**：若二重极限和两个累次极限都存在，则三者相等。
    * 注：累次极限存在意味着先横后纵（或先纵后横）接近 $P_0$。
* **结论推广**：闭区域连续、最值定理、介值定理等性质可参考一元函数。

## 二、 偏导数与全微分

### 1. 偏导数 (Partial Derivatives)
* **记号**：$f'_x(x_0, y_0)$, $\frac{\partial z}{\partial x}\big|_{P_0}$, $\frac{\partial f}{\partial x}$。
* **定义**：也就是关于 $x$ 的导数（将 $y$ 看作常数）。
  $$\lim_{\Delta x \to 0} \frac{f(x_0 + \Delta x, y_0) - f(x_0, y_0)}{\Delta x}$$
* **混合偏导数**：若二阶混合偏导数**连续**，则顺序无关（即 $f''_{xy} = f''_{yx}$）。
* **注意**：在分界点求偏导，不可“**先求导后代值**”，必须用**定义**求解。

### 2. 全微分 (Total Differential)
* **全增量公式**：
  $$\Delta z = A\Delta x + B\Delta y + o(\rho) \quad (\rho = \sqrt{(\Delta x)^2 + (\Delta y)^2} \to 0)$$
  其中 $A = f'_x, B = f'_y$。
* **全微分形式**：
  $$dz = \frac{\partial z}{\partial x}dx + \frac{\partial z}{\partial y}dy$$
* **可微与偏导的关系**：
    * 可微 $\implies$ 偏导存在：若在某点可微，则该点两个偏导数必存在。
    * 偏导存在 $\not\implies$ 可微：反之不一定成立。
* **不可微的判定**：若极限
  $$\lim_{\rho \to 0} \frac{\Delta z - (A\Delta x + B\Delta y)}{\rho}$$
  不存在或不为 $0$，则函数不可微。
* **充分条件**：若两个偏导数在区域内**连续**，则函数在该区域**可微**。

### 3. 复合函数求导（链式法则）
若 $z = f(u, v)$，且 $u = \varphi(x, y), v = \psi(x, y)$，则：
$$\frac{\partial z}{\partial x} = \frac{\partial z}{\partial u} \cdot \frac{\partial u}{\partial x} + \frac{\partial z}{\partial v} \cdot \frac{\partial v}{\partial x}$$
$$\frac{\partial z}{\partial y} = \frac{\partial z}{\partial u} \cdot \frac{\partial u}{\partial y} + \frac{\partial z}{\partial v} \cdot \frac{\partial v}{\partial y}$$

## 三、 隐函数求导

### 1. 隐函数存在定理
设 $F(x, y, z)$ 具有连续偏导数，且 $F(P_0) = 0$。
* **关键条件**：若 $F'_z(P_0) \neq 0$，则方程在 $P_0$ 邻域内能唯一确定隐函数 $z = f(x, y)$。

### 2. 求导公式
$$\frac{\partial z}{\partial x} = -\frac{F'_x}{F'_z}, \quad \frac{\partial z}{\partial y} = -\frac{F'_y}{F'_z}$$

### 3. 方程组情形 (Jacobian / Cramer's Rule)
对于方程组 $\begin{cases} F(x, y, u, v) = 0 \\ G(x, y, u, v) = 0 \end{cases}$，确定 $u, v$ 为 $x, y$ 的函数。
需满足雅可比行列式 $J \neq 0$：
$$J = \frac{\partial(F, G)}{\partial(u, v)} = \begin{vmatrix} F_u & F_v \\ G_u & G_v \end{vmatrix} \neq 0$$
则偏导数为：
$$\frac{\partial u}{\partial x} = -\frac{1}{J} \frac{\partial(F, G)}{\partial(x, v)} = -\frac{\begin{vmatrix} F_x & F_v \\ G_x & G_v \end{vmatrix}}{\begin{vmatrix} F_u & F_v \\ G_u & G_v \end{vmatrix}}$$
*(推导过程本质上是利用全微分形式解二元一次方程组)*

## 四、 方向导数与梯度

### 1. 方向导数
函数在点 $P_0$ 沿方向 $l$ 的变化率。设 $l$ 的方向角为 $\alpha, \beta, \gamma$（方向余弦）。
$$\frac{\partial u}{\partial l}\bigg|_{P_0} = \frac{\partial u}{\partial x}\cos \alpha + \frac{\partial u}{\partial y}\cos \beta + \frac{\partial u}{\partial z}\cos \gamma$$

### 2. 梯度 (Gradient)
* **定义**：
  $$\text{grad } u = \left(\frac{\partial u}{\partial x}, \frac{\partial u}{\partial y}, \frac{\partial u}{\partial z}\right) = \frac{\partial u}{\partial x}\mathbf{i} + \frac{\partial u}{\partial y}\mathbf{j} + \frac{\partial u}{\partial z}\mathbf{k}$$
* **关系**：方向导数等于梯度与方向单位向量的点积。梯度的方向是函数增长最快的方向，梯度的模是最大变化率。
* **几何意义**：梯度垂直于等位面 ($u(x, y, z) = C$)。

## 五、 多元函数的极值

### 1. 泰勒公式 (Taylor's Formula)
二元函数的麦克劳林公式 ($n=1$ 时)：
$$f(x_0 + h, y_0 + k) \approx f(x_0, y_0) + \left(h\frac{\partial}{\partial x} + k\frac{\partial}{\partial y}\right)f$$
这也是二元函数的拉格朗日中值公式的形式。

### 2. 极值判别
* **必要条件**：极值点处梯度为 $0$（即 $f'_x = 0, f'_y = 0$），该点称为**驻点**。
* **充分条件 (极值判别法)**：
    记 $A = f''_{xx}(P_0), B = f''_{xy}(P_0), C = f''_{yy}(P_0)$。
    令 $\Delta = B^2 - AC$ *(注意：有的教材记为 $AC - B^2$，判别结论相反，此处按笔记内容整理)*：
    * $B^2 - AC < 0$：有极值。
        * 若 $A > 0$ (或 $C > 0$)：**极小值**。
        * 若 $A < 0$ (或 $C < 0$)：**极大值**。
    * $B^2 - AC > 0$：不是极值（鞍点）。
    * $B^2 - AC = 0$：判别法失效，需进一步研究。

### 3. 最值求法
**步骤**：
1. 求区域内部的所有驻点和偏导数不存在的点。
2. 求区域边界上的最值（常转化为条件极值）。
3. 比较上述所有点的函数值。

### 4. 拉格朗日乘数法 (Lagrange Multipliers)
* **解决条件极值问题**：在约束 $G(x, y, z) = 0$ 下求 $u = f(x, y, z)$ 的极值。
* **构造拉格朗日函数**：
  $$L(x, y, z, \lambda) = f(x, y, z) + \lambda G(x, y, z)$$
* **求解方程组**：
  $$\begin{cases} f_x + \lambda G_x = 0 \\ f_y + \lambda G_y = 0 \\ f_z + \lambda G_z = 0 \\ G(x, y, z) = 0 \end{cases}$$
* **本质**：极值点处目标函数的梯度与约束曲面的法向量平行 ($\nabla f \parallel \nabla G$)。

## 六、 几何应用

### 1. 曲面的切平面与法线
对于隐式方程给出的曲面 $S: F(x, y, z) = 0$。
* **法向量 $\mathbf{n}$**：即该点的梯度向量。
  $$\mathbf{n} = (F'_x, F'_y, F'_z)|_{P_0}$$
* **切平面方程**：
  $$F'_x(x - x_0) + F'_y(y - y_0) + F'_z(z - z_0) = 0$$
* **法线方程**：
  $$\frac{x - x_0}{F'_x} = \frac{y - y_0}{F'_y} = \frac{z - z_0}{F'_z}$$

### 2. 空间曲线的切线与法平面
曲线 $\Gamma$ 由两个曲面交线定义：
$$\begin{cases} F(x, y, z) = 0 \\ G(x, y, z) = 0 \end{cases}$$
* **切向量 $\boldsymbol{\tau}$**：两个曲面法向量的叉积。
  $$\boldsymbol{\tau} = \mathbf{n}_1 \times \mathbf{n}_2 = \text{grad } F \times \text{grad } G$$
* **切线方程**：利用切向量 $\boldsymbol{\tau} = (m, n, p)$。
  $$\frac{x - x_0}{m} = \frac{y - y_0}{n} = \frac{z - z_0}{p}$$
* **法平面方程**：
  $$m(x - x_0) + n(y - y_0) + p(z - z_0) = 0$$
